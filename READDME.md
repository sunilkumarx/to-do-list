# To-Do List Application (Python)

A simple command-line To-Do List application built using Python.  
This project is beginner-friendly and demonstrates basic Python concepts.

## Features
- Add new tasks
- View all tasks
- Delete tasks
- Menu-driven interface
- Input validation

## Technologies Used
- Python 3

## How to Run
1. Download or clone the repository
2. Open the project folder
3. Run the following command:

   python main.py

## Project Purpose
This project was created to practice Python basics and prepare for internships.
